# SkyHealth-Weather-wellness
Weather and wellness dashboard with real-time forecasts, voice search, and alerts.
SkyHealth – Smart Weather & Wellness App

 ## Setup Instructions
  1. Get a free API key from OpenWeatherMap
  2. Open script.js
  3. Replace this line:
const API_KEY = "YOUR_API_KEY"
with your own API key.

Features
- Real-time weather data
- Hourly forecast
- 7-day forecast
- Voice search
- Location detection
- Weather wellness tips
- Search history
- Popular cities quick search

Technologies
- HTML
- CSS
- JavaScript
- OpenWeather API
- Web Speech API
- Geolocation API

How to Run
1. Download project
2. Open index.html
3. Enter city name
